Details and features

Keyboard interaction

  • Use TAB key to enter the calendar. Focus is placed on the day that is selected. If no date is selected, first day in the month gets focus instead.
  • LEFT ARROW and RIGHT ARROW to move between days.
  • Use UP ARROW and DOWN ARROW to move between weeks
  • Select a date using SPACE or RETURN keys
  • Leave the calendar with TAB or SHIFT + TAB
